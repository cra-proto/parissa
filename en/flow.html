Global Minimum Tax (v2)

Who must file a GloBE information return

Answer interactive questions

To determine which entities must file a GIR with the CRA, use the following interactive questions.

A text version is available if needed.

If the interactive questions do not appear, view the text version.

Choose the options that apply to your MNE group and scroll down for more questions and results.

Does the MNE group have a designated filing entity located in Canada?

  • Yes, a designated filing entity is located in Canada.
  • No, there is no designated filing entity located in Canada.

    Is the MNE group's ultimate parent entity located in Canada?

    • Yes, the ultimate parent entity is located in Canada.
    • No, the ultimate parent entity is not located in Canada.

      Does the MNE group have a qualifying foreign filing entity?

      • Yes, there is a qualifying foreign filing entity.

        Did the CRA notify the MNE group that it has not received a GIR from the qualifying foreign filing entity’s jurisdiction?

        • Yes, the CRA notified the MNE group that it has not received a GIR from the qualifying foreign filing entity’s jurisdiction.
        • No, the CRA has not notified the MNE group that it has not received a GIR from the qualifying foreign filing entity’s jurisdiction.
      • No, there is no qualifying foreign filing entity.

        Does the MNE group have an ultimate parent entity or designated filing entity outside Canada that filed the GIR directly with the CRA on or before the GIR due date?

        • Yes, an ultimate parent entity or designated filing entity outside Canada has filed the GIR directly with the CRA on or before the GIR due date.
        • No, neither an ultimate parent entity nor a designated filing entity outside Canada has filed the GIR directly with the CRA on or before the GIR due date.

Based on your selections above:

The designated filing entity located in Canada must file the GIR.

Based on your selections above:

The ultimate parent entity located in Canada must file the GIR.

Based on your selections above:

Each constituent entity located in Canada or a single appointed constituent entity must file the GIR.

Based on your selections above:

No constituent entity in Canada needs to file the GIR.

View text version

Use this version if you do not want to answer interactive questions.

One of the following constituent entities of a qualifying MNE group must file a GloBE information return with the CRA:

  • The designated filing entity must file when there one that is located in Canada
  • The ultimate parent entity must file when the following conditions are all met:
    • The ultimate parent entity is located in Canada
    • The MNE group does not have a designated filing entity located in Canada
  • Each constituent entity located in Canada or a single appointed constituent entity must file when the following conditions are all met:
    • The MNE group does not have an ultimate parent entity or designated filing entity located in Canada
    • The MNE group does not have an ultimate parent entity or designated filing entity outside Canada that filed the GIR directly with the CRA on or before the GIR due date
    • Either the MNE group does not have a qualifying foreign filing entity or it has a qualifying foreign filing entity, but the CRA notified the MNE group that it has not received a GIR from the qualifying foreign filing entity's jurisdiction
